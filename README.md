# Event Market Analysis

Apple-style React market event page with Stock and ETF analysis modules.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## GitHub Pages

This app is configured to deploy with GitHub Actions. After pushing to the `main` branch of `emeryxu1-blip/event-market-analysis`, GitHub Pages will publish the built site at:

```text
https://emeryxu1-blip.github.io/event-market-analysis/
```
